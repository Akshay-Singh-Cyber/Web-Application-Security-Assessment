# Screenshots

## Overview

This directory is intended to contain screenshots demonstrating the web application security assessment workflow and the use of common security testing tools.

To follow responsible disclosure practices, screenshots containing organization-specific information, vulnerability details, or sensitive technical data are **not included** in this public repository.

Examples of suitable screenshots include:

- Burp Suite interface (sanitized)
- Nmap interface
- Nikto scan example
- Browser Developer Tools
- OWASP Top 10 workflow diagrams
- Generic TLS/SSL analysis examples
