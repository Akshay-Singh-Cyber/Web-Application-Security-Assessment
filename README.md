This repository showcases the methodology, workflow, tools, and documentation practices followed during an **authorized academic web application security assessment**.

The assessment followed the **OWASP Top 10 (2021)** methodology and involved reconnaissance, vulnerability identification, manual verification, risk assessment, and remediation planning.

To follow **responsible disclosure practices**, organization-specific findings, technical proof-of-concepts, and sensitive assessment data have been intentionally excluded from this public repository.
